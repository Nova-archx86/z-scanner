# Z-scan
A port scanner inspired by nmap (https://nmap.org) written in zig

# Installation

### Building from source

Requirements:
1. zig 0.10.1

Clone the repo using git and build the project using zig build
    
    git clone https://github.com/Nova-archx86/zscan
    
    cd zscan; zig build
